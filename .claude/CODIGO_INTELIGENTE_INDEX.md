## 🎓 Conceptos Clave

### Algoritmo Inteligente
```
Descripción: "Corte recto de precisión en tela"
    ↓
Extrae palabras: ["Corte", "Recto", "Precisión", "Tela"]
    ↓
Genera abreviatura: "CORT-RECT"
    ↓
Recomienda nombre: "Corte Recto"
    ↓
Resultado: Código + Nombre automáticos
```

### Modos de Abreviatura
```
'corto'    → "CR" (primera letra cada palabra)
'largo'    → "CORT-RECT" (primeros 4 chars)
'1-char'   → "C" (exactamente 1 char)
'2-char'   → "CR" (exactamente 2 chars)
'3-char'   → "COR" (exactamente 3 chars)
'4-char'   → "CORT" (exactamente 4 chars)
```

### Formatos Soportados
```
Servicios: XXX-XXX-NNN     (descripción + atributos)
MP:        TT-SS-COL-DDDD  (4 atributos específicos)
PT:        G-TT-F-COL-DDDD (5 atributos específicos)
```

## 📞 Preguntas Frecuentes

**P: ¿Funciona en Next.js 16?**  
A: Sí, usa React 19 + Server/Client components nativamente.

**P: ¿Necesito cambiar la BD?**  
A: No, el sistema genera códigos/nombres automáticamente. La BD almacena lo de siempre.

**P: ¿Y si tengo un formato especial?**  
A: Puedes combinar funciones. Ver code-generator.README.md para casos especiales.

**P: ¿Funciona sin descripción?**  
A: Sí, usa abreviaturas configuradas o genera desde el nombre del atributo.

**P: ¿Puedo desactivar las recomendaciones?**  
A: Sí, configurando abreviaturas manuales completas en los atributos.
